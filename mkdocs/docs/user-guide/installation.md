## Installation

1. Clone the repository:

```bash
git clone https://github.com/kreativermario/SkyWizz.git
```

2. Install the dependencies:

```bash
pip install -r requirements.txt
```

3. Run the bot, make sure you are in the correct folder.
```bash
python SkyWizz.py
```
or
```bash
python3 SkyWizz.py
```
